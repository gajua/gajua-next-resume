# 포트폴리오 웹사이트 프로젝트
[포트폴리오 웹사이트](https://gajua-next-resume-khaki.vercel.app/)

## 프로젝트 소개
Next.js 13과 최신 웹 기술을 활용하여 개발한 개인 포트폴리오 웹사이트입니다. 사용자 경험(UX)을 최우선으로 고려하여 설계하였으며, 웹 접근성과 인터랙티브한 요소들을 통해 차별화된 포트폴리오를 구현하였습니다.

## UX 중심 설계 및 구현 사항

### 1. 사용자 인터랙션 최적화
- **스크롤 기반 애니메이션**
  - Framer Motion을 활용한 부드러운 스크롤 인터랙션
  - 스크롤 위치에 따른 자연스러운 요소 전환
  - 성능 최적화를 위한 `will-change` 속성 활용

- **마우스 움직임 추적**
  - 배경 요소들의 시차(Parallax) 효과 구현
  - requestAnimationFrame을 활용한 부드러운 움직임
  - 디바이스 성능을 고려한 애니메이션 최적화

### 2. 접근성 중심 개발
- **키보드 네비게이션 개선**
  - 모든 상호작용 요소에 키보드 포커스 지원
  - Enter/Space 키 이벤트 처리로 동등한 사용성 제공
  - 포커스 순서 최적화 및 시각적 피드백 강화

- **스크린리더 지원**
  - WAI-ARIA 규칙을 준수한 시맨틱 마크업
  - 동적 콘텐츠 변경 시 적절한 피드백 제공
  - 장식용 요소와 실제 콘텐츠의 명확한 구분

### 3. 사용자 피드백 시스템
- **폼 인터랙션**
  - 실시간 유효성 검사 및 피드백
  - 오류 메시지의 명확한 시각적 표현
  - 제출 상태에 대한 즉각적인 피드백

- **상태 표시**
  - 로딩 상태의 스켈레톤 UI 구현
  - 작업 완료 시 성공 애니메이션 제공
  - 오류 발생 시 복구 방법 안내

### 4. 성능 최적화
- **이미지 최적화**
  - WebP 형식 사용으로 로딩 속도 개선
  - Next.js Image 컴포넌트를 활용한 자동 최적화
  - 적절한 이미지 크기 및 품질 설정

- **코드 최적화**
  - React 컴포넌트 렌더링 최적화
  - 불필요한 리렌더링 방지
  - 코드 스플리팅을 통한 초기 로딩 최적화

### 5. 반응형 디자인
- **유동적 레이아웃**
  - 다양한 디바이스에 최적화된 UI
  - 브레이크포인트별 레이아웃 조정
  - 터치 인터랙션 지원

- **다크모드**
  - 시스템 설정 연동
  - 부드러운 테마 전환 효과
  - 접근성을 고려한 색상 대비

## 기술 스택
- **Frontend**: Next.js 13, TypeScript, TailwindCSS
- **애니메이션**: Framer Motion
- **UI 컴포넌트**: shadcn/ui
- **이메일 통합**: EmailJS
- **배포**: Vercel

## 성과 및 학습
- 웹 접근성 표준(WCAG)을 준수한 개발 경험 획득
- 사용자 중심의 UI/UX 설계 능력 향상
- 성능 최적화 및 디버깅 스킬 향상
- 최신 웹 기술 스택 활용 능력 강화
